# Choosing Common Plots

## RULES

Always check the tool registry to choose an appropriate tool first. If an appropriate tool exists, use that tool. Use code generation ONLY when an appropriate tool does not exist in th tool registry.

## Histogram
Use when you have one numeric variable and want to inspect distribution shape, skewness, spread, or outliers.

## Bar chart
Use when you have one categorical variable and want to compare counts or proportions across groups.

## Correlation heatmap
Use when you have multiple numeric variables and want to inspect pairwise linear relationships.

## Missingness plot
Use when you want to understand data quality and compare missing-data rates across variables.

## Boxplot
Use when plotting a numeric variable across groups
Common choices:
boxplot for distributions by category
violin plot for distribution shape
bar chart of means only if summary is desired

## Practical rule
Start simple. A small number of clear exploratory plots is usually more helpful than many crowded figures.
